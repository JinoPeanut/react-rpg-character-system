# React RPG Character System

간단한 RPG 캐릭터 능력치 시스템을 React와 Zustand로 구현한 프로젝트입니다.
레벨업, 스탯 분배, 전직 시스템, 파생 능력치 계산 등을 구현했습니다.

## 기술스택

- React
- Zustand
- Vite
  
## 기능

- 레벨업 시 포인트 추가

  캐릭터 레벨업 시 스탯 포인트가 지급됩니다.

- 캐릭터 스탯 시스템

  STR, DEX, INT, LUK 4가지 기본 능력치를 관리합니다.

- 스탯 포인트 분배

  레벨업으로 얻은 포인트를 원하는 스탯에 투자할 수 있습니다.

- 스탯 초기화

  투자한 스탯 포인트를 초기 상태로 되돌릴 수 있습니다.

- 스탯 요구 조건 기반 전직

  일정 레벨과 스탯 조건을 만족하면 직업 전직이 가능합니다.

- 직업 시스템

  기본 직업: 모험가

  전직 가능 직업

  전사

  궁수

  마법사

  도적

- 파생 능력치 계산 (Derived Stats)

  공격력 / 마력은 캐릭터 스탯을 기반으로 계산됩니다.

  직업마다 서로 다른 계산 공식이 적용됩니다.

- 체력(HP) 계산 시스템

  직업별 기본 체력과 레벨 성장 수치를 기반으로 계산됩니다.

  일부 직업은 특정 스탯에 따라 추가 체력 보너스를 받을 수 있습니다.

- 직업별 능력치 공식

  HP / 공격력 / 마력 / 방어력 와 같은 수치는 캐릭터의 기본적인 스탯을 기반으로 계산하여 밸런싱 했습니다.

- 직업별 기본 크리티컬

  직업마다 다른 크리티컬 확률과 기본적으로 계산되는 크리티컬 데미지 2배 적용하여 추가적인 계산 포함 할 예정입니다.

## 시스템 특징

- Zustand를 사용하여 캐릭터 상태를 전역 관리

- 직업 데이터 기반 능력치 계산 구조

- 스탯 변경 시 파생 능력치 자동 업데이트

## 앞으로 추가 예정

- 치명타 확률(Crit) 계산 시스템

- 추가 파생 능력치 (방어력, 회피 등)

- 장비(Equipment) 시스템

- 캐릭터 능력치 UI 스타일 개선
